# Usage Data
Provides a way to track how much profiling data your project are sending to the Cloud. This is useful to get an idea of how much data you are sending to the Cloud and how much it costs.

## Current Usage

The Current Usage section shows the amount of daily data your projects have sent to the Polar Signals Cloud during the current billing cycle. This usage is broken down by the data from various projects in your organization.


You can find this in the billing section of the organization settings page.

![Current Usage](/img/billing-usage-metrics.png)

:::info
Note: If you are on a trial plan, then this section will show data for the current calendar month.
:::
